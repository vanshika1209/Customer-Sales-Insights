# 📊 Customer Sales Insights

## 📌 Overview
Customer Sales Insights is a data analytics project that examines customer purchasing behavior, revenue trends, and the effectiveness of discounts. Using SQL for analysis and Power BI for visualization, this project uncovers patterns that help businesses make smarter, data-driven decisions.

## 🎯 Objective
The main goal of this project is to analyze customer data to:
- Understand revenue distribution
- Identify high-value and loyal customers
- Measure the impact of discounts on sales
- Compare behavior across different customer segments
- Provide actionable business recommendations

## 🛠️ Tools & Technologies
- SQL (PostgreSQL)
- Power BI
- Jupyter Notebook
- CSV Dataset

## 📂 Dataset
The dataset includes customer-level shopping data with fields such as:
customer_id, gender, age, city, purchase_amount, discount_applied, subscription_status, shipping_type, previous_purchases, category, item_purchased, and review_rating.

## 🔍 Analysis Summary
The project includes multiple SQL-based analyses such as:
- Revenue comparison across gender
- Identifying customers who spend more even with discounts
- Finding top-rated products
- Comparing purchase behavior by shipping type
- Analyzing subscriber vs non-subscriber spending
- Customer segmentation (New, Returning, Loyal)
- Identifying top products within categories

## 📊 Dashboard
An interactive Power BI dashboard was created to visualize:
- Revenue distribution
- Product performance
- City-wise sales trends
- Customer segments
- Discount impact
- Subscription behavior

## 💡 Key Insights
- Discounted customers still contribute significantly to revenue  
- Loyal customers generate the highest overall value  
- High-rated products tend to perform better  
- Subscription-based customers spend more on average  

## 🚀 Business Recommendations
- Strengthen loyalty programs to retain valuable customers  
- Use discounts strategically instead of broadly  
- Promote high-performing products  
- Encourage repeat customers to subscribe  
- Apply personalized marketing based on customer segments  

## 📌 Conclusion
This project demonstrates how combining SQL and Power BI can uncover meaningful insights from customer data and support better business decision-making.
